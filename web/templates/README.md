# Templates

HTML шаблоны для веб-интерфейса.

## Структура

- **layouts/** - Базовые шаблоны (header, footer, base layout)
- **auth/** - Страницы аутентификации (login)
- **errors/** - Страницы ошибок (404, 500)
- **users/** - Страницы управления пользователями
- **groups/** - Страницы управления группами
- **exchanges/** - Страницы управления биржами
- **positions/** - Страницы торговых позиций
- **market/** - Страницы рыночного анализа
- **coins/** - Страницы управления монетами
